# trabalho_4bi
Sistema de Controle de Despesas

Este projeto é um sistema simples de controle de despesas desenvolvido em Java. Ele permite registrar despesas, anotar pagamentos e realizar outras operações relacionadas ao gerenciamento financeiro pessoal.
Funcionalidades
Registrar novas despesas
Registrar pagamentos
Exibir resumo de despesas
Menu interativo no terminal
Estrutura orientada a objetos com classes e abstrações
Classes Principais
Classe	Descrição
Main	Controla a execução do programa e exibe o menu principal
Despesa	Classe abstrata para representar uma despesa, exigindo implementação do cálculo de imposto
Usuario	Representa um usuário do sistema, armazenando login e senha criptografada (a implementação atual ainda usa a senha diretamente)
